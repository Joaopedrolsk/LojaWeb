# LojaWeb

> Uma aplicação web moderna para gerenciamento de lojas online, desenvolvida com Java, CSS e JavaScript.

[![Status](https://img.shields.io/badge/status-ativo-brightgreen)]()
[![Linguagem Principal](https://img.shields.io/badge/Java-74.4%25-orange)]()
[![Licença](https://img.shields.io/badge/licença-MIT-blue)]()

## 📋 Descrição

LojaWeb é uma solução completa para gerenciamento de lojas virtuais, oferecendo funcionalidades robustas e uma interface intuitiva para vendedores e clientes. O projeto combina o poder do backend em Java com uma experiência frontend responsiva utilizando CSS e JavaScript.

## 🎯 Características Principais

- ✅ Gestão de produtos e catálogo
- ✅ Carrinho de compras intuitivo
- ✅ Sistema de autenticação e autorização
- ✅ Processamento de pedidos
- ✅ Interface responsiva e moderna
- ✅ Escalabilidade e desempenho otimizado

## 🛠️ Tecnologias Utilizadas

### Backend
- **Java** (74.4%) - Lógica de negócio e API REST
- Frameworks e bibliotecas Java

### Frontend
- **CSS** (22.1%) - Estilização e responsividade
- **JavaScript** (3.5%) - Interatividade e validações

## 📦 Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- [Java JDK](https://www.oracle.com/java/technologies/javase-downloads.html) (versão 11+)
- [Maven](https://maven.apache.org/download.cgi) ou [Gradle](https://gradle.org/install/)
- Um navegador web moderno
- [Git](https://git-scm.com/)

## 🚀 Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/Joaopedrolsk/LojaWeb.git
cd LojaWeb
```

### 2. Configure o ambiente

```bash
# Configure as variáveis de ambiente necessárias
# Consulte o arquivo .env.example ou a documentação do projeto
```

### 3. Compile e execute

```bash
# Com Maven
mvn clean install
mvn spring-boot:run

# Com Gradle
gradle build
gradle bootRun
```

### 4. Acesse a aplicação

Abra seu navegador e acesse: `http://localhost:8080`
